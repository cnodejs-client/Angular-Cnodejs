# Angular-Cnodejs
基于Angular.js重写Cnodejs.org社区的webapp



## 目录结构

>
.
├── README.md           
├── dist               // 项目build目录
├── index.html         // 项目入口文件
├── package.json       // 项目配置文件
├── src                // 生产目录
│   ├── assets         // css js 和图片资源
│   ├── components     // 各种组件
│   ├── views          // 各种页面
│   ├── filters.js     // 各种过滤器
│   └── main.js        // gulp 预编译入口
├── app.js             // 服务配置
└── config.js          //   配置文件
